# Franmaker

Franmaker es una aplicación SaaS para la creación y gestión de franquicias. Esta aplicación incluye una página de inicio, un sistema de precios y un formulario de login.

## Características

- Página de inicio
- Sistema de precios con diferentes planes
- Formulario de login

## Tecnologías utilizadas

- React
- React Router
- Tailwind CSS

## Despliegue

Esta aplicación está configurada para ser desplegada directamente en Vercel.
